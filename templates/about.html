IoT Home Automation and Security System

DomSafe is a comprehensive IoT-based home automation and security system built on Raspberry Pi. It integrates real-time environmental monitoring, security features, and smart device control into a unified platform.

This system demonstrates the practical application of Internet of Things technologies for residential security and automation, featuring motion detection, RFID authentication, environmental sensing, and remote device control.

üîó System Architecture

1. Raspberry Pi collects sensor data and manages hardware components

2. MQTT Client sends real-time data to Adafruit IO for live monitoring

3. Local SQLite stores data when internet is unavailable

4. Cloud Database (Neon PostgreSQL) syncs and stores historical data

5. Flask Web App provides user interface and device control

6. HTTP API fetches live data from Adafruit IO and historical data from cloud database
